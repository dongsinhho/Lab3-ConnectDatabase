# **Bài thực hành số 2**

Bài thực hành được viết bằng nodejs và template engine PUG trên framework ExpressJS. Database sử dụng là mongodb chạy trên cloud Mongo Atlas 

Khởi động App bằng lệnh: `npm run dev`
```
    App chạy ở localhost:3000 
```
